# 이미지를 테스트
## Ⅰ. 테스트 대상 2가지
React에서 이미지를 테스트 할게 2가지 있습니다.  
ⅰ. 이미지가 존재하는지?  
ⅱ. attribute 값이 올바른지?

### ⅰ. 이미지가 존재하는지?
만약 `import`로 이미지를 불렀는데 파일이 없다면 빌드가 실패하게 됩니다.  
따라서 이럴때는 이미지 존재 여부를 테스트 하지 않구요.

[toBeVisible()](https://github.com/testing-library/jest-dom#tobevisible)를 써서 실제 유저에게 보이는지를 테스트합니다.  
img element가 `hidden` 속성이나 css 등으로 숨겨질 경우가 있기 때문입니다.  
ex)
```js
describe('Img', () => {
  it('img is visible', () => {
    const { getByRole } = render(<Image />);

    expect(getByRole('img')).toBeVisible();
  });
});
``` 

### ⅱ. attribute 값이 올바른지?
img에 들어갈 대표적인 속성 값인 `src`나`alt` 값이 올바른지 테스트합니다.  
ex)
```js
describe('Img', () => {
  it('img contains correct attribute values', () => {
    const { getByRole } = render(<Image />);

    expect(getByRole('img')).toHaveAttribute('src', 'images/blabla.png');
    expect(getByRole('img')).toHaveAttribute('alt', 'blabla');
  });
});
```
이렇게 2가지 테스트 방법으로 이미지를 테스트합니다.
