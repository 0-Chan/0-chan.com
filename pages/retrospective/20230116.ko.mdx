## Coding Mates

Coding Mates라는 사이드 프로젝트에 합류했다.  
기획과 백엔드 개발이 완료되어 프론트엔드 개발만이 남아있는 프로젝트였다.  
들어보니 기존 프론트엔드 개발자가 연락 두절이 되는 바람에 새로 구하게 되었다고 한다.

나로서는 더할 나위 없이 정말 좋은 기회였는데,  
와이어프레임과 API 스펙이 모두 나온 상태라 개발에만 집중할 수 있기 때문이었다.

## 개발 환경 구축

처음에는 그래도 기존에 작업하던 repository가 있으니 이를 이어서 작업할까 했었는데.  
Next.js가 13이 나왔는데 12를 쓰고 있기도 했고 무엇보다 진척 사항이 거의 없어서 새로 개발 환경을 구축하기로 했다.

그래도 storybook을 사용하는 방법 등의 레퍼런스를 얻을 수 있었다.
